# Доки итераций

Каждая итерация `N` из `ROADMAP.md` при закрытии (`/iterationClose N`) кладёт сюда папку
`docs/iterations/NN/` (двузначный номер) с тремя файлами:

- **`explainer.md`** — что итерация делает, простым языком; мини-ликбез по каждой новой
  концепции через её питоний аналог (правило 2 конституции); поток данных псевдографикой;
  карта «где в коде». Читатель — будущий-ты, зашедший через полгода.
- **`demo.md`** — happy-path смок-тест: пронумерованные копипаст-исполнимые шаги, под каждым
  «Ожидаемо». Этот файл церемония **прогоняет вживую**, а не только пишет.
- **`learnings.md`** — принятые решения и почему; грабли (особенно «где Node повёл себя не
  как Python»); сознательные срезы и отложенные TODO (`dw-lite:`-маркеры).

Находки код-ревью сюда не переносятся — исключение только для сознательно принятого решения
«что-то НЕ чинить» (оно живёт в `learnings.md` как проектное решение).
