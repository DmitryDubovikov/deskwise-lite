showcase-workflow)»**. Контроль анти-дрейфа: *«каждая практика north-star → строка таблицы

## Питоний аналог

## Решения (2026-07-23, спрошено)
README — **English** (резюме-артефакт; русские доки остаются в `docs/iterations/`);
**demo-PR** с красным contract-drift gate создаётся и остаётся открытым (как authwise PR #1);
**live OpenAI** для кадров стриминга/summarize — ок (~$0.01).

## Кадры (чек-лист ROADMAP «Витрина» → носитель по принципам памяти)
1. Zod-схема роута ↔ Orval-хук рядом — **код-скрин** (HTML-рендер, два пейна).
2. Правка Zod-поля → `tsc` в web падает — **GIF** (терминальные кадры).
3. Contract-drift gate поймал рассинхрон — **скрин красного PR** (GitHub UI) + живая ссылка.
4. UI: список / деталь / переход статуса — **скрины браузера** (playwright).
5. Токены suggest-reply дорисовываются в UI — **GIF** (live) + `curl -N` — **fenced text**.
6. `docker compose ps` + nginx-конфиг (статика, `/api/`, SSE-локация) — **fenced text**.
7. pino JSON-лог с request-id — **fenced text**.

- `README.md` (English) в структуре витрины: hero → таблица north-star-практик, каждая
  строка ссылается на секцию с доказательством → Run it → резюме-строки.
- Все 7 кадров существуют в `docs/assets/` и вставлены в README; GIF'ов 2 (кадры 2 и 5).
- Demo-PR открыт, required check красный, README ссылается на него.
- Резюме-строки и стек-строка — только из реально продемонстрированного (красная линия

+ ревью-пайплайн чист (CRITICAL/BUG = 0).

## Шаги
1. Поднять стек (`make up` + seed + live-ключ), подготовить съёмочную технику в скретчпаде
   (venv + playwright, ffmpeg).
2. Снять кадры 1–2, 4–7 (стек локально); залить в `docs/assets/`.
3. Demo-ветка `demo/contract-drift` (правка Zod-схемы без регенерации) → PR → красный CI →
   скрин; PR остаётся открытым.
4. Написать showcase-README (структура authwise, адаптированная под героя deskwise:
   schema-first → e2e types → CI gate → AI/SSE); текущий Quickstart переехать в «Run it».
5. Ревью-пайплайн (general + constitution → аудитор → фиксы) + `/simplify` не по коду, а
   финальная вычитка диффа README.
